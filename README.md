# HTML Auxiliary Generator

## 简要介绍

### HTML 辅助生成工具

- 辅助前端开发人员更轻松更简单的完成范围性 HTML 视图的实现

- 使用方式就和低代码拖拽开发一样，只不过这款 web 应用的定位点是作为辅助工具而不是全盘代替前端视图的开发，
  当你遇到一些较为复杂的 html 视图，或者只想偷懒不想费心思手写，那么你可以选择打开此工具，拖住组合一下就能生成对应的 html+css 代码，cv 过来即可。
